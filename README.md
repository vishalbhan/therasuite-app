TheraSuite - a comprehensive mental health practice management platform for therapists, psychologists, and coaches. Here
are the key features:

Core Features

🗓️ Appointment Management
- Weekly calendar with drag-and-drop scheduling
- Multiple session types (video, in-person, hybrid)
- Automated email confirmations and reminders
- Session lengths: 30-120 minutes

👥 Client Management
- Complete client profiles and history
- Visual client cards with avatars
- Session tracking and filtering

💻 Video Conferencing
- Integrated Dyte video platform
- Seamless therapist/client video sessions
- Real-time notes during calls

📝 Session Documentation
- Real-time note-taking during sessions
- Comprehensive session history
- Mandatory post-session documentation

💰 Billing & Invoicing
- Multi-currency support (INR, USD, EUR, GBP, AUD, CAD)
- Automated invoice generation and email delivery
- Payment tracking (Pending → Invoice Sent → Payment Received)

📊 Dashboard Analytics
- Today's sessions and weekly hours
- Active client count and monthly revenue
- Calendar integration with quick actions

⚙️ Settings & Configuration
- Professional profile setup
- Payment method configuration
- Session preferences and location management

🔐 Security & Authentication
- Supabase authentication
- Row-level security for data privacy
- HIPAA-compliant design patterns

The app is built with React/TypeScript, Supabase, and modern tools like Tailwind CSS and React Query, providing a complete end-to-end
  solution for running a therapy practice.
